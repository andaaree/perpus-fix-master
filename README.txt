# Requirements :
* Server :
* 1. Php 7.4.27
* 2. MySQL 10.4.22-MariaDB
* 3. Composer 2.2.* (local)
* 4. Laravel 6.20.*

* Extensions:
* 1. Yajra\DataTable
* 2. Ghostscript 9.5.50
* 3. Resumable.js 2.*
* 4. Guzzle\Http

# First Setup :
1. Run " composer install "
2. Run " php artisan migrate:fresh --seed "
3. Run " php artisan key:generate "
